Word Definition
pixels: Computer based images are comprised of thousands of tiny dots of color
Resolution: indicates the number of pixels that are displayed per inch for an image (or pixels per centimeter)
lossy compression: the class of data encoding methods that uses inexact approximations and partial data discarding to represent the content
lossless compression: is a class of data compression algorithms that allows the original data to be perfectly reconstructed from the compressed data.
JPEG: Joint Photographic Expert Groups
PNG: Portable Network Graphics
SVG: Scalable Vector Graphics
